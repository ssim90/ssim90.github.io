REQUIREMENTS
This module requires the following modules:
- assets (It includes video files and button images)
- css (It includes css file for the button)
- javascript (It includes babylon js engine javascipt library and source codes for the module)
- index.html

INSTALLATION
1. Create a bucket in amazon s3.
2. Add files mentioned above to the bucket.
3. Change 'Access' setting of the bucket to public.
4. From the object list, select all the objects, choose actions, and then choose 'Make public'.
5. Now, you can access the site with html Object URL ( Click index.html and find Object URL )
